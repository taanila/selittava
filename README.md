# Selittävän analytiikan perusmenetelmät

Voit ladata esimerkit omalle koneelle: Napsauta **Code** (vihreä pudotusvalikko-painike) ja valitse **Download ZIP**. Pura ZIP-paketti itsellesi sopivaan hakemistoon.

Esimerkkejä aiheista ristiintaulukointi kategorisille muuttujille, tilastollisten tunnuslukujen vertailu kategorisen muuttujan määräämissä ryhmissä, korrelaatiot, khiin neliö -testi, t-testi, korrelaatiokertoimen testaus.

Lisätietoa Pythonin käytöstä data-analytiikassa: https://tilastoapu.wordpress.com/python
